-------------------------------------------------------------------------------

public-API approach end-to-end before any polish. (ref: virtual-desktops.md)
  [x] Add VirtualDesktopInfo { id: vdi } in main.qml (read currentDesktop,

      "current" property (UUID). (legacy setCurrentDesktop(int) fallback: virtual-desktops.md)

  [x] WorkspaceDot: property (active) + MouseArea -> activated()

  [x] Advertise size: implicitWidth/Height from dot count * size + spacing
VERIFY: dots count == numberOfDesktops; clicking a dot switches; switching via
        keyboard (Ctrl+F1/F2) updates the active dot (reactive binding works).


-------------------------------------------------------------------------------
 MILESTONE 2 — GNOME visual identity: dots + sliding pill                [ ]
-------------------------------------------------------------------------------
Goal: the signature look — dim circles + a wider pill on the current desktop that
slides between positions. Horizontal panel first. (ref: kirigami.md, qml-performance.md)
  [ ] Inactive dot: Rectangle radius=h/2, Kirigami.Theme.textColor @ ~0.45 opacity
  [ ] Active pill: single overlay Rectangle (rounded, wider) over the active dot
  [ ] Bind pill x/width to the active dot geometry; Behavior on x/width (NumberAnimation)
      for the slide — duration from Kirigami.Units (guard 0 when animations disabled)
  [ ] Pill color = Kirigami.Theme.highlightColor (theme-following by default)
  [ ] Size everything via Kirigami.Units (HiDPI); no hardcoded px
VERIFY: matches GNOME feel; pill slides smoothly on switch; follows color scheme;
        looks correct on HiDPI / fractional scaling.

 MILESTONE 3 — Interactions                                              [ ]
-------------------------------------------------------------------------------
Goal: scroll, hover, add/remove. Read config keys directly (defaults from main.xml
already apply even before the settings UI exists). (ref: virtual-desktops.md)
  [ ] Scroll-to-switch: MouseArea onWheel -> step(+/-1); honor enableScroll + scrollWrap;

  [ ] Hover: dot brightens on containsMouse
  [ ] Tooltips: show desktop name (vdi.desktopNames) on hover; honor showTooltips

      wired to createDesktop / removeDesktop DBus; honor enableAddRemove; never remove
      the last desktop
VERIFY: scroll cycles (wrap per setting); hover highlights + tooltip; right-click
        add/remove updates dot count live.
